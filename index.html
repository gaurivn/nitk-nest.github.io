NeST: Network Stack Tester

Perform tests for different congestion control algorithms and queue disciplines.

Python

Users need not learn new programming languages or syntax to use the tool. It lets the user write all the code necessary for the experimenting in python.

group

All in one tool

NeST is a python package that handles testbed setup, testbed configuration, collecting and visualizing data by providing a user friendly API, addressing common issues involved in conducting networking experiments.

computer

Minimal requirements

The tool doesn't need high system requirements to be able to run it. It just needs a Linux Operating System with netperf version 2.7 and iperf version 3.0.11 or above installed.



About the Project

We are a team of students from NITK Surathkal working on this project under the guidance of Dr. Mohit P. Tahiliani. The main idea is to use Linux network namespaces and provide APIs to create complex emulated networks, run tests and extract the statistics using iproute2 and netperf in a single python script.
